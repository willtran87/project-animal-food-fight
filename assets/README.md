# Root Asset Notes

Root-level assets are reserved for shared browser chrome and cross-entry files that do not belong to a single runtime art family.

- `favicon-192.png` - active browser favicon used by `index.html` and the local test pages.
- `favicon-source-chromakey-v1.png` - generated favicon source on a flat chroma-key background, retained for provenance.
- `favicon-transparent-v1.png` - transparent favicon intermediate used to package the active favicon.
